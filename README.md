# ProjectMaze

Проект-прототип на Unity, изначально начатый как основа небольшой игры с процедурно генерируемым лабиринтом.

Текущая наиболее развитая часть проекта — многоэтапная система процедурной генерации карты: создание базового лабиринта, размещение комнат, стохастические модификации структуры, автоматическое соединение ключевых областей и анализ достижимости.

## Текущий статус

**Статус:** восстановленный рабочий прототип.

Проект был поднят после длительного перерыва и мигрирован на **Unity 6.3 LTS**.

На момент последней ревизии:

- проект успешно открывается и запускается;
- процедурная генерация работает;
- управление игроком работает;
- проект переведён на нормальную структуру Git-репозитория;
- основная ветка — `main`;
- генерируемые Unity- и IDE-файлы исключены из Git;
- внешняя среда разработки — Visual Studio Code.

## Технологии

- Unity 6.3 LTS
- C#
- Unity Tilemap
- Unity Input System
- Rigidbody2D
- Cinemachine

В алгоритмической части используются:

- рандомизированная генерация лабиринта семейства Prim;
- A*;
- BFS;
- Fisher–Yates shuffle;
- стохастические преобразования структуры.

# Текущий pipeline генерации

При запуске сцены основной процесс запускается из:

`MazeGeneratorAlg.Start()`

Общая последовательность:

```text
1. Очистка Tilemap
        ↓
2. Создание центральной стартовой комнаты
        ↓
3. Размещение игрока
        ↓
4. Размещение дополнительных комнат
        ↓
5. Заполнение оставшегося пространства стенами
        ↓
6. Генерация базового лабиринта
        ↓
7. Добавление кластеров стен
        ↓
8. Случайное расширение коридоров
        ↓
9. Оформление и подключение комнат
        ↓
10. Удаление одиночных стен
        ↓
11. Размещение и подключение выхода
        ↓
12. Секторный проход коррекции
        ↓
13. BFS-анализ достижимости
        ↓
14. Финальная укладка пола
